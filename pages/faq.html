Foire aux questions

Retrouvez ici les réponses aux questions les plus posées par les utilisateurs. Vous ne trouvez pas la réponse à votre question ? Utilisez le formulaire de contact pour m'envoyer un message !

A quoi servent les scripts proposés ?

Les Scripts présentés ici servent à ajouter/modifier le contenu de vos serveurs GTA RP. Ils vous permettent de personnaliser votre expérience sans limite.

Comment installer et déployer le script sur mon serveur ?

Une fois téléchargé via le lien proposé, rien de plus simple :

* Décompressez le dossier et placez le dans votre dossier RESSOURCE
* Mettre toutes les lignes dans [Nom_de_la_ressource].sql dans la base de données.
* Dans Serveur.cfg, ajoutez : start [Nom_de_la_ressource]

Puis-je demander un script en particulier ?

Si vous ne trouvez pas le script que vous souhaitez installer, n'hésitez pas à me contacter sur les réseaux sociaux ou via le formulaire prévu à cet effet. Je prendrais alors directement contact avec vous pour discuter de vos attentes.

Il y a une erreur dans le script que j'ai téléchargé. Comment faire ?

En cas d'erreur, vous pouvez directement m'envoyer un message via le formulaire prévu à cet effet. Si vous le pouvez, précisez le type d'erreur rencontré et indiquez un maximum de détails, je prendrais contact avec vous dans les meilleurs délais.
